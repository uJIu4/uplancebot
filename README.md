# Uplancebot
Telegram bot for job notifications

Add me in telegram: [Uplancebot]

Set a query for search -> Get a messages with new jobs by your query

### Commands

- /start - let's start
- /change - change a query
- /query - what your query
- /feedback - write your feedback
- /help - get help
